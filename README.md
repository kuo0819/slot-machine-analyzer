🎰 AI 趨勢戰略家 (AI Trend Strategist - Steel Will Edition)
🚀 這是什麼？
這是一個專為電子遊戲（Slots）設計的即時 RTP 監測與注額管理工具。它不保證中獎，但它能透過數學邏輯與「強制冷靜機制」，保護你的本金不受到情緒波動（上頭）的傷害。

🛠 核心功能
1. 數據化趨勢分析
即時 RTP 計算：根據每 100 轉的餘額變化，精算出機台當前的返還率（RTP）。

注額建議邏輯：

極凍區 (RTP < 70%)：強烈建議縮注，保留體力。

洗盤區 (RTP 90-110%)：維持基準注額，等待吐分期。

大熱區 (RTP > 130%)：趨勢升溫，加注進攻。

2. 鋼鐵意志 - 防上頭系統 (Anti-Tilt)
這是本工具最強大的特色，專門對付「情緒化追注」：

強制冷靜鎖定：當總虧損達到初始本金的 50%，系統會啟動「死鎖模式」。

確認鈕禁用：倒數 60 秒，期間無法點擊。

重置鈕同步鎖定：你無法透過清空數據來規避懲罰。

跨頁面記憶：就算重新整理網頁，鎖定時間依舊持續，逼你必須休息。

視覺心理警示：虧損嚴重時，介面將切換為「深紫色脈衝閃爍」，抑制大腦的博弈興奮感。

3. 進度管理
動態損益進度條：視覺化顯示你離「止損線」與「止盈目標」的距離。

隨機執行語錄：在計算時隨機顯示警語，刺破賭徒幻覺。

📖 使用指南
設定基準值：進入頁面後，先輸入你的 初始本金、目標獲利 以及 基準單注。

數據錄入：

點擊 「100轉前餘額」，輸入當前餘額。

手動轉動遊戲 100 轉後，點擊 「100轉後餘額」 並輸入新數字。

執行建議：點擊「確認」，觀察系統給出的 $ 建議注額。

嚴守紀律：

如果系統變紫色並進入倒數，請放下手機，離開座位洗臉或喝水。

當達到目標獲利（進度條滿格），請執行物理提款。
